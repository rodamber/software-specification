## Import project

1. Open RODIN.
2. Click on "File -> Import...".
3. Select "Existing Projects into Workspace" and click "Next".
4. Make sure "Select archive file" is selected and click "Browse...".
5. Select Zober-G04-P1-1617.zip.
6. Click "Finish".
